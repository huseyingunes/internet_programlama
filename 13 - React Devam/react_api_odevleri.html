React & API Entegrasyonu Ödev Listesi

Ödev #1

İnsan Kaynakları (HR) Dashboard

Büyük bir teknoloji şirketi için çalışan yönetim paneli tasarlayın. API'den gelen verileri kullanarak sanal bir şirket kadrosu oluşturun.
Kullanılacak API:
API Ninjas (Random User API) Ödev Gereksinimleri:
  • API'den 10-15 arası rastgele kullanıcı çekin.
  • Her kullanıcı için şık bir 'Personel Kartı' (Card UI) tasarlayın.
  • İsim ile anlık arama (search) yapılabilecek bir filtreleme sistemi kurun.
  • Kullanıcı kartlarına 'Sil' butonu ekleyerek state üzerinden listeden çıkarma işlemi yapın.
Odaklanılacak Konular: useState, useEffect, Filter/Map methods
Ödev #2

Kişiselleştirilmiş Haber Bülteni

Kullanıcının ilgi alanlarına göre kategorize edilmiş haberleri okuyabileceği modern bir haber portalı geliştirin.
Kullanılacak API:
NewsAPI Ödev Gereksinimleri:
  • Kategori sekmeleri oluşturun (Teknoloji, Bilim, Sağlık vb.).
  • Seçilen kategori değiştikçe API'ye yeni istek atılmasını sağlayın.
  • Haberleri görsel, başlık ve kısa açıklama ile listeleyin.
  • Haberin orijinal sitesine gitmek için 'Devamını Oku' linki ekleyin.
Odaklanılacak Konular: Dynamic API Fetching, Dependency Array
Ödev #3

Geliştirici API Portalı

Geliştiricilerin proje fikirleri bulabileceği bir API kataloğu uygulaması hazırlayın.
Kullanılacak API:
PublicAPI.dev Ödev Gereksinimleri:
  • Tüm API listesini çekin ve tablo veya grid yapısında sunun.
  • API'leri 'Auth (Var/Yok)' ve 'HTTPS' durumuna göre filtreleyin.
  • Rastgele bir API getiren bir 'İlham Ver' butonu tasarlayın.
  • Boş arama sonuçları için 'API Bulunamadı' mesajı gösterin.
Odaklanılacak Konular: Object destructuring, Conditional Rendering
Ödev #4

Moodify: Moduna Göre Müzik

Kullanıcının o anki ruh haline göre (Enerjik, Melankolik, Odaklanmış) çalma listeleri öneren bir uygulama.
Kullanılacak API:
Spotify API Ödev Gereksinimleri:
  • Mod seçim butonları tasarlayın.
  • Seçilen moda uygun türdeki (genre) şarkıları listeleyin.
  • Şarkı albüm kapaklarını ve sürelerini düzgün bir formatta gösterin.
  • Basit bir 'Beğen' butonu ile seçilen şarkıları favori listesine ekleyin.
Odaklanılacak Konular: Props Management, Array Operations
Ödev #5

DevBreak: Yazılımcı Şakaları

Yazılımcıların mola anlarında stres atabileceği, etkileşimli bir şaka uygulaması.
Kullanılacak API:
Official Joke API Ödev Gereksinimleri:
  • Bir şaka sorusu (setup) gösterin.
  • 'Cevabı Gör' butonuna basılınca cevabı (punchline) animasyonla açın.
  • 'Yeni Şaka Getir' butonu ile asenkron veri akışını yönetin.
  • Şakalara 1-5 arası puan verebileceğiniz bir oylama sistemi ekleyin.
Odaklanılacak Konular: Async/Await, UI State Management
Ödev #6

VocabUp: Kelime Kartları

İngilizce öğrenenler için kapsamlı bir sözlük ve kelime çalışma aracı.
Kullanılacak API:
Wordnik API Ödev Gereksinimleri:
  • Kelime araması yapılabilecek bir form oluşturun.
  • API'den kelimenin tanımı, telaffuzu ve örnek cümlelerini çekin.
  • Eş anlamlı kelimeleri (synonyms) tıklanabilir linkler olarak sunun.
  • Aranan kelimeleri 'Son Arananlar' listesinde tutun.
Odaklanılacak Konular: Controlled Inputs, Multi-fetch handling
Ödev #7

Chef's Table: Tarif Rehberi

Mutfaktaki malzemelere göre yemek tarifleri sunan bir şef asistanı.
Kullanılacak API:
TheMealDB API Ödev Gereksinimleri:
  • Malzeme bazlı veya kategori bazlı arama özelliği ekleyin.
  • Yemek tarifini modal (açılır pencere) içerisinde gösterin.
  • Tarifin içindeki malzeme listesini 'checkbox' listesi olarak sunun.
  • Tarif videosu varsa bir embed video alanı ekleyin.
Odaklanılacak Konular: Complex JSON parsing, Modals/Portals
Ödev #8

Cosmos Explorer: NASA Günü

NASA'nın her gün yayınladığı eşsiz uzay fotoğraflarını keşfeden bir takvim uygulaması.
Kullanılacak API:
NASA APOD API Ödev Gereksinimleri:
  • Uygulama açılışında 'Günün Astronomi Fotoğrafı'nı gösterin.
  • Tarih seçici (Date Input) ile geçmiş tarihlerdeki fotoğraflara gidin.
  • Fotoğraf yüksek çözünürlükte yüklenirken bir 'Loading' animasyonu gösterin.
  • Fotoğrafın açıklama metnini okunabilir bir tipografi ile sunun.
Odaklanılacak Konular: Date formatting, Loading states
