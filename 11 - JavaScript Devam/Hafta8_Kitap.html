Bölüm 01

Array Metodları — Fonksiyonel Veri Dönüşümü

Döngü yazmak yerine veriyi dönüştürün, filtreleyin, tek değere indirin. ES5'ten bu yana var olan bu metodlar, modern JavaScript'in en temel yapı taşlarıdır.

📜
Fonksiyonel Programlama Kökü

map, filter ve reduce kavramları 1950'lerdeki Lisp dilinden gelir. John McCarthy'nin Lisp'i (1958), fonksiyonu birinci sınıf vatandaş olarak tanıyan ilk dildi. Haskell (1990) bu yaklaşımı zirveye taşıdı. ES5 (2009) bu üç metodu JavaScript'e kazandırdı; ES6 (2015) arrow function ile kullanımı dramatik biçimde sadeleştirdi.

map — Her Elemanı Dönüştür

map, bir diziyi alır ve her elemanı bir fonksiyondan geçirerek aynı uzunlukta yeni bir dizi döndürür. Orijinal dizi değişmez:

JavaScript
const sayilar = [1, 2, 3, 4, 5];

// Karelerini al
const kareler = sayilar.map(x => x * x);
// [1, 4, 9, 16, 25]

// Nesne dizisinden alan çıkarma
const kullanicilar = [
  { ad: 'Ahmet', yas: 25, aktif: true },
  { ad: 'Ayşe',  yas: 30, aktif: false },
  { ad: 'Mehmet', yas: 22, aktif: true },
];

const adlar = kullanicilar.map(k => k.ad);
// ['Ahmet', 'Ayşe', 'Mehmet']

// Dönüşüm — her kullanıcıya label ekle
const etiketli = kullanicilar.map(k => ({
  ...k,
  label: `${k.ad} (${k.yas})`
}));

// React'ta liste render:
// kullanicilar.map(k => <li key={k.ad}>{k.ad}</li>)

filter — Koşulu Sağlayanları Al

filter, her eleman için bir test fonksiyonu çalıştırır; true döndürenleri yeni diziye alır. Uzunluk azalabilir, hiç değişmeyebilir:

JavaScript
const sayilar = [1,2,3,4,5,6,7,8];

const ciftler = sayilar.filter(x => x % 2 === 0);
// [2, 4, 6, 8]

// Aktif kullanıcıları bul
const aktifler = kullanicilar.filter(k => k.aktif);
// [{ad:'Ahmet',...}, {ad:'Mehmet',...}]

// filter + map zincirleme — sık kullanılan kalıp
const aktifAdlar = kullanicilar
  .filter(k => k.aktif)
  .map(k => k.ad);
// ['Ahmet', 'Mehmet']

// Falsy değerleri temizle
const temiz = [1, null, 'a', undefined, 0, 'b'].filter(Boolean);
// [1, 'a', 'b']

reduce — Tek Değere İndir

reduce en güçlü ve en esnek array metodudur. Bir birikimli değer (acc) ve mevcut eleman (curr) alır; sonuç herhangi bir tip olabilir — sayı, nesne, dizi, string:

JavaScript
// Basit toplam
const toplam = [1,2,3,4,5].reduce((acc, curr) => acc + curr, 0);
// 15

// Sepet toplamı — gerçek dünya
const sepet = [
  { urun: 'Kitap',   fiyat: 45, adet: 2 },
  { urun: 'Kalem',   fiyat: 5,  adet: 10 },
  { urun: 'Defter',  fiyat: 20, adet: 3 },
];
const genelToplam = sepet.reduce((acc, item) =>
  acc + item.fiyat * item.adet, 0);
// 180  (90 + 50 + 60)

// Gruplama — şehire göre
const sehirler = kullanicilar.reduce((acc, k) => {
  const sehir = k.sehir ?? 'Bilinmiyor';
  (acc[sehir] ??= []).push(k.ad);
  return acc;
}, {});
// { 'İstanbul': ['Ahmet'], 'Ankara': ['Ayşe','Mehmet'] }

// reduce ile map yapımı (kavramsal)
const myMap = (arr, fn) => arr.reduce((acc, x) => [...acc, fn(x)], []);
// map ve filter, reduce ile yazılabilir

flat ve flatMap

JavaScript
// flat — iç içe dizileri düzleştir
[[1,2],[3,4],[5]].flat();
// [1, 2, 3, 4, 5]  — 1 seviye (varsayılan)

[1, [2, [3, [4]]]].flat(Infinity);
// [1, 2, 3, 4]  — tüm derinlikler

// flatMap = map + flat(1) — çok yaygın!
const cumleler = ['merhaba dünya', 'iyi günler'];
cumleler.flatMap(s => s.split(' '));
// ['merhaba', 'dünya', 'iyi', 'günler']

// API yanıtlarında — kategoriler içinde ürünler
const kategoriler = [
  { ad: 'Elektronik', urunler: ['Telefon', 'Laptop'] },
  { ad: 'Giyim',      urunler: ['Gömlek', 'Pantolon'] },
];
const tumUrunler = kategoriler.flatMap(k => k.urunler);
// ['Telefon', 'Laptop', 'Gömlek', 'Pantolon']

Zincirleme ve Karşılaştırma

MetodGirdiÇıktıAmaç

filter[a,b,c]≤ girdi uzunluğu diziKoşulu sağlayanları seç
reduce[a,b,c]herhangi bir tipTek değere indir
flat[[a],[b,c]]düz diziİç içeyi düzleştir
flatMap[a,b]düz dizimap + flat(1)

✦
Zincirleme Kalıbı

Array metodları zincirlenerek kullanılır. Okunaklı, test edilebilir ve fonksiyonel: veri.filter(koşul).map(dönüşüm).reduce(birleşim). React'ta liste render etmek için her zaman map kullanın — forEach değer döndürmez, JSX ile çalışmaz.

Gerçek Dünya: Sipariş Analizi

JavaScript — E-ticaret Raporu
const siparisler = [
  { id: 1, musteri: 'Ahmet', urun: 'Laptop',  fiyat: 15000, teslim: true  },
  { id: 2, musteri: 'Ayşe',  urun: 'Telefon', fiyat: 8000,  teslim: false },
  { id: 3, musteri: 'Ahmet', urun: 'Klavye',  fiyat: 500,   teslim: true  },
  { id: 4, musteri: 'Mehmet',urun: 'Monitör', fiyat: 5000,  teslim: false },
];

// 1. Teslim edilmemiş siparişlerin toplamı
const bekleyen = siparisler
  .filter(s => !s.teslim)
  .reduce((acc, s) => acc + s.fiyat, 0);
// 13000

// 2. Müşteri bazında özet
const musteriOzet = siparisler.reduce((acc, s) => {
  acc[s.musteri] ??= { toplam: 0, adet: 0 };
  acc[s.musteri].toplam += s.fiyat;
  acc[s.musteri].adet++;
  return acc;
}, {});
// { Ahmet: {toplam:15500, adet:2}, Ayşe: {...}, Mehmet: {...} }

// 3. En pahalı ürünlerin adı
const pahaliUrunler = siparisler
  .filter(s => s.fiyat > 5000)
  .map(s => s.urun);
// ['Laptop', 'Telefon']
Bölüm 02

Object Metodları, Spread ve Destructuring

Nesneleri döngüde gezmek, kopyalamak, birleştirmek ve parçalarını çıkarmak için modern JavaScript'in sunduğu güçlü araçlar.

Object.keys, Object.values, Object.entries

JavaScript
const urun = {
  ad:     'Laptop',
  fiyat:  15000,
  stok:   50,
  aktif:  true,
};

Object.keys(urun);
// ['ad', 'fiyat', 'stok', 'aktif']

Object.values(urun);
// ['Laptop', 15000, 50, true]

Object.entries(urun);
// [['ad','Laptop'], ['fiyat',15000], ['stok',50], ['aktif',true]]

// Nesneyi döngüde gez
Object.entries(urun).forEach(([anahtar, deger]) => {
  console.log(`${anahtar}: ${deger}`);
});

// Nesneyi filtrele (Object.fromEntries ile geri dön)
const sayisallar = Object.fromEntries(
  Object.entries(urun).filter(([, v]) => typeof v === 'number')
);
// { fiyat: 15000, stok: 50 }

Spread Operatörü (...)

Spread, bir nesneyi veya diziyi "açarak" yeni bir yapıya kopyalar. Sığ kopya (shallow copy) oluşturur:

JavaScript
const temel = { x: 1, y: 2 };

// Kopyala
const kopya = { ...temel };           // { x:1, y:2 }

// Genişlet
const genis = { ...temel, z: 3 };   // { x:1, y:2, z:3 }

// Güncelle — React state kalıbı
const durum = { ad: 'Ali', yas: 25, sehir: 'Ankara' };
const guncellendi = { ...durum, yas: 26 };
// { ad:'Ali', yas:26, sehir:'Ankara' } — orijinal değişmedi

// Birleştir — sağdaki öncelikli
const varsayilan = { renk: 'mavi', boyut: 'M', aktif: true };
const ozel       = { renk: 'kırmızı' };
const sonuc = { ...varsayilan, ...ozel };
// { renk:'kırmızı', boyut:'M', aktif:true }

// Dizi spread
const a = [1,2], b = [3,4];
const c = [...a, ...b, 5]; // [1, 2, 3, 4, 5]
⚠
Sığ Kopya Uyarısı

Spread, yalnızca bir seviye kopyalar. İç içe nesneler hâlâ aynı referansı paylaşır. const a = {x: {y: 1}}; const b = {...a}; b.x.y = 2; yaptığınızda a.x.y de 2 olur. Derin kopya için structuredClone(obj) kullanın.

Rest Operatörü (...)

JavaScript
// Nesne rest — kalanları topla
const { ad, yas, ...geri } = kullanici;
// geri = { sehir:'İstanbul', aktif:true }

// Belirli alanları çıkarmak için
const { sifre, ...guvenli } = kullanici;
// API yanıtını şifresiz gönder

// Fonksiyon rest parametresi
function topla(ilk, ...digerleri) {
  return digerleri.reduce((acc, n) => acc + n, ilk);
}
topla(1, 2, 3, 4); // 10

// Dizi rest
const [ilk, ikinci, ...kalan] = [10, 20, 30, 40, 50];
// ilk=10, ikinci=20, kalan=[30,40,50]

Destructuring — Derinlemesine

JavaScript
// Varsayılan değer
const { ad, sehir = 'Bilinmiyor' } = kullanici;

// Yeniden adlandırma
const { ad: kullaniciAdi, yas: kullaniciYasi } = kullanici;

// İç içe destructuring
const kullanici2 = {
  ad: 'Zeynep',
  adres: { sehir: 'İzmir', ilce: 'Bornova' }
};
const { ad: isim, adres: { sehir, ilce } } = kullanici2;
// isim='Zeynep', sehir='İzmir', ilce='Bornova'

// Fonksiyon parametresinde destructuring
function kullaniciyiGoster({ ad, yas, sehir = 'Bilinmiyor' }) {
  console.log(`${ad}, ${yas}, ${sehir}`);
}
kullaniciyiGoster({ ad: 'Ali', yas: 30 });
// 'Ali, 30, Bilinmiyor'

// Dizi swap
let [a, b] = [1, 2];
[a, b] = [b, a]; // a=2, b=1  — geçici değişkensiz!
Bölüm 03

ES Modülleri — import / export

Her dosya izole bir kapsama sahip olur. Global kirlilik ortadan kalkar, kod organize edilir, bağımlılıklar açık hale gelir.

📜
Modül Sistemleri Tarihi

ES6'dan önce JavaScript'te resmi modül sistemi yoktu. 2009 — Node.js, CommonJS'i (require/module.exports) tanıttı. 2011 — AMD (RequireJS), tarayıcı için asenkron modül yükleme getirdi. 2015 — ES6, import/export ile dil düzeyinde modül sistemini standartlaştırdı. Bugün tarayıcılar ve Node.js ikisini de destekler; modern projeler hep ES modülü kullanır.

Named Export

matematik.js
// Named export — her dosyada istediğiniz kadar
export function topla(a, b)  { return a + b; }
export function carp(a, b)  { return a * b; }
export function bol(a, b)   { return a / b; }
export const PI = 3.14159265;
export const E  = 2.71828182;

// veya sonda gruplu export:
export { topla, carp, bol, PI, E };
app.js — import
// Belirli isimleri import et (destructuring benzeri)
import { topla, PI } from './matematik.js';

// Hepsini namespace altında
import * as mat from './matematik.js';
mat.topla(2, 3); // 5
mat.PI;          // 3.14159...

// Yeniden adlandırma
import { topla as ekle } from './matematik.js';

Default Export

JavaScript
// kullanici.js — bir dosyada yalnızca 1 default export
export default function kullaniciyiGetir(id) {
  return fetch(`/api/users/${id}`).then(r => r.json());
}

// app.js — istediğiniz isimle import
import getUser from './kullanici.js';   // ✓
import fetchUser from './kullanici.js'; // ✓ aynı şey

// Default + named birlikte
import kullaniciyiGetir, { dogrula, guncelle } from './kullanici.js';

Barrel Pattern (index.js)

Büyük projelerde çok fazla import satırı oluşur. Barrel pattern, bir klasörün tüm export'larını tek noktadan sunar:

utils/index.js
// utils/index.js — tek çıkış noktası
export { topla, carp } from './matematik.js';
export { formatTarih } from './tarih.js';
export { dogrula }    from './validasyon.js';

// app.js — tek import
import { topla, formatTarih, dogrula } from './utils';
// utils/index.js otomatik okunur

Dinamik import ve Code Splitting

JavaScript
// Statik import — derleme zamanında yüklenir
import { buyukKutuphane } from './buyuk.js';

// Dinamik import — çalışma zamanında, gerektiğinde
btn.addEventListener('click', async () => {
  const { grafik } = await import('./grafik.js');
  grafik.ciz(veri);
});
// grafik.js yalnızca butona tıklandığında indirilir
// → ilk yükleme hızlanır (code splitting)

Bölüm 04

DOM — Document Object Model

Tarayıcı HTML'yi bir ağaç yapısına (DOM) çevirir. JavaScript bu ağacı okuyabilir, değiştirebilir, yeni düğümler ekleyebilir veya silebilir. Her değişiklik ekrana yansır.

Element Seçiciler

JavaScript
// Tek element — CSS seçici
const btn   = document.querySelector('#gonder');
const h1    = document.querySelector('h1');
const kart  = document.querySelector('.kart');
const input = document.querySelector('form input[type="email"]');

// Tüm eşleşenler — NodeList (dizi benzeri)
const tumBtn = document.querySelectorAll('button');
tumBtn.forEach(btn => btn.classList.add('stilum'));

// Eski API — hâlâ geçerli
document.getElementById('id');
document.getElementsByClassName('cls'); // canlı HTMLCollection

// Bir element içinde arama
const form = document.querySelector('#giris-form');
const emailInput = form.querySelector('input[type="email"]');

İçerik Manipülasyonu

JavaScript
const el = document.querySelector('#baslik');

// textContent — sadece metin, XSS güvenli
el.textContent = 'Yeni Başlık';

// innerHTML — HTML parse eder (dikkat: XSS riski!)
el.innerHTML = '<strong>Kalın</strong> başlık';

// Attribute
el.setAttribute('data-id', '42');
el.getAttribute('data-id'); // '42'
el.removeAttribute('hidden');

// dataset — data-* attribute'ları
el.dataset.kullaniciId = '123'; // data-kullanici-id="123"
el.dataset.kullaniciId;          // '123'
✕
innerHTML ve XSS

Kullanıcıdan gelen veriyi doğrudan innerHTML'e koymayın: el.innerHTML = kullaniciGirdisi XSS (Cross-Site Scripting) açığı oluşturur. Kullanıcı verisi için her zaman textContent kullanın veya DOMPurify gibi bir sanitizer kullanın.

Stil ve Sınıf Manipülasyonu

JavaScript
const btn = document.querySelector('#btn');

// classList — önerilen yöntem
btn.classList.add('aktif');
btn.classList.remove('pasif');
btn.classList.toggle('acik');    // var ise kaldır, yok ise ekle
btn.classList.contains('aktif'); // true/false
btn.classList.replace('eski', 'yeni');

// Inline stil — çok kullanmayın (CSS'te tutun)
btn.style.backgroundColor = '#1E2761';
btn.style.display          = 'none';
btn.style.fontSize         = '16px';

// Hesaplanmış stil (salt okunur)
const styles = getComputedStyle(btn);
styles.marginTop; // '16px'

Element Oluşturma ve Ekleme

JavaScript
// Yeni element oluştur
const li = document.createElement('li');
li.textContent = 'Yeni Öğe';
li.classList.add('liste-item');
li.dataset.id = '42';

// Ekleme yöntemleri
const ul = document.querySelector('ul');
ul.appendChild(li);         // sona ekle
ul.prepend(li);             // başa ekle
ul.insertBefore(li, hedef); // hedeften önce
ul.replaceChild(yeni, eski); // değiştir

// Modern: insertAdjacentHTML
ul.insertAdjacentHTML('beforeend', '<li>Hızlı yol</li>');

// Silme
li.remove();                 // elementi sil
ul.removeChild(li);         // eski yöntem

Event Delegation

Her elemente ayrı listener eklemek yerine ortak bir ataya tek listener bağla, event.target ile olayı başlatanı bul:

✕ Yanlış — N listener
// 100 ürün = 100 listener
document.querySelectorAll('.sil')
  .forEach(btn =>
    btn.addEventListener('click', sil)
  );
// Dinamik eklenenler çalışmaz!
✓ Doğru — 1 listener
liste.addEventListener('click', e => {
  if (!e.target.matches('.sil'))
    return;
  const id = e.target.dataset.id;
  urunSil(id);
});
// Dinamik eklenenler de çalışır!

DocumentFragment — Performanslı DOM

Her appendChild çağrısı tarayıcıda bir reflow tetikler. 100 eleman ekleyecekseniz 100 reflow yerine tek reflow:

JavaScript
const urunler = [/* 1000 ürün */];
const liste   = document.querySelector('#urun-listesi');

// Yavaş — her appendChild reflow tetikler
urunler.forEach(u => {
  const li = document.createElement('li');
  li.textContent = u.ad;
  liste.appendChild(li); // 1000 reflow!
});

// Hızlı — DocumentFragment (DOM dışı tampon)
const frag = document.createDocumentFragment();
urunler.forEach(u => {
  const li = document.createElement('li');
  li.textContent = u.ad;
  frag.appendChild(li); // DOM'a dokunmaz
});
liste.appendChild(frag); // tek reflow ✓
Bölüm 05

AJAX ve fetch API

Sayfa yenilemeden sunucuyla konuşmak. 1999'daki XMLHttpRequest'ten 2015'teki fetch API'ye uzanan yolculuk.

📜
AJAX'ın Tarihi

1999 — Microsoft, Outlook Web Access için XMLHttpRequest'i geliştirdi. 2004 — Google Maps ve Gmail, XHR'ı geniş kitlelere tanıttı ve "web uygulaması" kavramını doğurdu. 2005 — Jesse James Garrett bu tekniğe "AJAX" (Asynchronous JavaScript and XML) adını koydu. 2015 — ES6 ile fetch API geldi, Promise tabanlı modern standart. 2017 — async/await fetch'i daha da okunabilir kıldı.

XMLHttpRequest — Tarihsel Bağlam

JavaScript — Eski Yöntem
// XHR — artık kullanmıyoruz, ama nasıl çalıştığını bilin
const xhr = new XMLHttpRequest();
xhr.open('GET', 'https://api.example.com/users');
xhr.responseType = 'json';

xhr.onload = function() {
  if (xhr.status === 200) {
    console.log(xhr.response);
  } else {
    console.error('Hata:', xhr.status);
  }
};
xhr.onerror = () => console.error('Ağ hatası');
xhr.send();

// Sorunlar: callback hell, hata yönetimi zorlu,
// Promise desteği yok, sözdizimi karmaşık

fetch API — Modern Standart

JavaScript
// Temel fetch — Promise zinciri
fetch('https://jsonplaceholder.typicode.com/users')
  .then(yanit => {
    if (!yanit.ok) throw new Error(`HTTP ${yanit.status}`);
    return yanit.json();
  })
  .then(kullanicilar => console.log(kullanicilar))
  .catch(hata => console.error(hata));

// async/await — tercih edilen
async function kullanicilariGetir() {
  try {
    const yanit = await fetch('/api/users');
    if (!yanit.ok) throw new Error(`HTTP ${yanit.status}`);
    const veri  = await yanit.json();
    return veri;
  } catch (e) {
    console.error('Hata:', e.message);
  }
}

// Diğer response metodları
yanit.text();   // düz metin
yanit.blob();   // ikili veri (resim vb.)
yanit.headers.get('Content-Type');
⚠
fetch Hata Yakalamaz!

fetch, 404 veya 500 yanıtlarında reject etmez — sadece ağ hatalarında reject eder. Bu yüzden if (!yanit.ok) throw new Error(...) kontrolü şarttır. Bu, XHR'dan gelen en önemli fark ve en sık yapılan hatadır.

POST, PUT, DELETE İstekleri

JavaScript
// Genel fetch wrapper — tüm projelerde kullanılabilir
async function apiFetch(url, options = {}) {
  try {
    const yanit = await fetch(url, options);

    if (yanit.status === 401) throw new Error('Oturum açın');
    if (yanit.status === 403) throw new Error('Yetki yok');
    if (yanit.status === 404) throw new Error('Bulunamadı');
    if (!yanit.ok)             throw new Error(`HTTP ${yanit.status}`);

    return yanit.json();
  } catch (e) {
    if (e.name === 'TypeError') // ağ hatası
      throw new Error('Sunucuya ulaşılamadı');
    throw e; // diğer hataları ilet
  }
}

Tam Örnek — fetch + DOM + Array Metodları

JavaScript — Kullanıcı Listesi Uygulaması
// jsonplaceholder.typicode.com — ücretsiz test API
async function uygulamaBaslat() {
  const liste = document.querySelector('#kullanici-listesi');
  const arama = document.querySelector('#arama');

  // 1. Veriyi çek
  const kullanicilar = await fetch(
    'https://jsonplaceholder.typicode.com/users'
  ).then(r => r.json());

  // 2. DOM'a yaz (array metod + DocumentFragment)
  function listeyiRender(liste_verisi) {
    const frag = document.createDocumentFragment();
    liste_verisi.forEach(k => {
      const li = document.createElement('li');
      li.innerHTML = `<strong>${k.name}</strong> — ${k.email}`;
      li.dataset.id = k.id;
      frag.appendChild(li);
    });
    liste.innerHTML = '';
    liste.appendChild(frag);
  }

  listeyiRender(kullanicilar);

  // 3. Arama — filter + render
  arama.addEventListener('input', () => {
    const q = arama.value.toLowerCase();
    const filtrelendi = kullanicilar.filter(k =>
      k.name.toLowerCase().includes(q) ||
      k.email.toLowerCase().includes(q)
    );
    listeyiRender(filtrelendi);
  });

  // 4. Event delegation — tüm satırlara tek listener
  liste.addEventListener('click', async (e) => {
    const li = e.target.closest('li');
    if (!li) return;
    const id = li.dataset.id;
    const detay = await fetch(`/api/users/${id}`).then(r => r.json());
    console.log(detay);
  });
}

uygulamaBaslat();
